# DIY_3D_Printed_Electronic_CNC_Machine

If designed properly, 3D printed parts are strong enough to build and use a well functioning CNC machine. 
I used a 40% infill to print each parts out of PLA, although you could use other materials such a PETG. 
Above you can find STL files and STEP files for any further modifications of your own choice.
